# Project NeoWrite

NeoWrite ist ein graphisches Frontent für Neovim. Während man mein Neovim-config auch ohne graphische Oberfläche nutzen kann bietet die Kombination mit Neovide eine moderne und aesthetisch ansprechende Oberfläche für die Nutzung Neovims zur Erstellung von Markdownfiles und dessen Konvertierung.


## Work in progress

Folgendes ist für Project Neovide noch in Planung:

mit Dmenu/Rofi/... auf __Startseite__

Ferner:
+ Unterstützung verschiedener Tastaturenlayouts (Shortcuts)


## Dependencies

+ Neovim, Neovide
+ Plug (Plugin-manager)
+ Nerd-Fonts
