# Mein Desktop

Dies ist die Repository meiner Desktop-Oberfläche, basierend auf **BSPWM** für meine Linux-Systeme.
Betrachte dieses Projekt als 'Work in Process'. In nächster Zeit werden noch einige Ergänzungen hinzugefügt, wie beispielsweise das GTK-Theme, Paketlisten und Scripte.


## Die Idee

Das Konzept war das Erstellen einer simplen Nutzeroberfläche, welche auf Produktivität und Textbearbeitung mit so wenig Ablenkungfsfaktoren wie möglich ausgelegt ist. Das Projekt fußt dabei auf folgenden drei Säulen:

1. Simple Nutzeroberfläche
2. Leichtes Betriebssystem
3. Unix-Philosophie


## Benötigte Packete

Die folgenden Packete finden in dieser Desktop-Umgebung Gebrauch:

* BSPWM
* SXHKD
* Polybar
* Alacritty
* Nitrogen
* Neovim
* Nerdfonts
* Gnu-Stow ( > stow * -t $HOME )
* PCmanFM
* Firefox/Surf
* Rofi
